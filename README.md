## 🔧 What I'm building

🚀 **[studde](https://github.com/priyankaforu/studde)** — AI-powered German university application platform. Handles document verification, uni-assist workflows, and visa guidance for Indian students heading to Germany.

📦 **[JadaWorks](https://github.com/priyankaforu/jadaworks)** — Social media scheduler and content repurposing tool. Built with TypeScript.

🎨 **[PostMaker](https://github.com/priyankaforu/post-maker)** — Turn screenshots into shareable social media posts in seconds.

🍎 **[BestBelly](https://github.com/priyankaforu/BestBelly)** — Nutri-score app that provides detailed nutritional analysis of products. Also built a [Flutter mobile version](https://github.com/priyankaforu/bestbelly-mobile).

📖 **[Katha AI Book](https://github.com/priyankaforu/katha-AI-book)** — AI-powered book project (MVP stage).

🌐 **[vibeinpublic](https://github.com/priyankaforu/vibeinpublic)** — A community platform for solo builders shipping in public.

💬 **[JusConnect](https://github.com/priyankaforu/JusConnect-ChatApp)** — Real-time chat app for sharing texts and images between users.

🐦 **[X-Twitter Reply Counter](https://github.com/priyankaforu/X-Twitter-reply-counter)** — Counts replies on any Twitter/X account automatically in real time.

🔌 **[Twitter Extension](https://github.com/priyankaforu/Twitter-Extension)** — Browser extension for hiding likes and extracting posts with impressions.

🏪 **[Product Store](https://github.com/priyankaforu/product-store)** — Inventory management app for business owners.

🌍 **[Website-ITG](https://github.com/priyankaforu/Website-ITG)** — Website for "India To Germany" consulting firm.
